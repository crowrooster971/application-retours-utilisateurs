# Application Retours Utilisateurs

## Description
Cette application permet aux utilisateurs de partager leurs retours sur des produits et services. Les gestionnaires peuvent consulter, organiser et analyser les retours facilement, permettant ainsi d'améliorer les services offerts.

## Fonctionnalités
- Soumission facile de feedback par les utilisateurs
- Organisation des retours par catégorie, produit ou service
- Système de notifications pour les gestionnaires
- Analyse des retours via des graphiques et des statistiques
- Réponses des gestionnaires directement via l'interface

## Installation
1. Clonez le dépôt avec `git clone <url-du-repo>`.
2. Installez les dépendances avec `npm install`.
3. Démarrez l'application en utilisant `npm start`.
